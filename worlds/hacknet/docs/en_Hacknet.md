# Hacknet

## Where is the settings page?

The [player settings page for this game](../player-settings) contains all the options you need to configure and export a config file.

## What is a "check" in Hacknet?

* Completing missions for Entropy, CSEC, or Bibliotheque
* Completing the /el Sec path
* Certain one-off events - such as completing the Kaguya Trials
* Gaining achievements - OPTIONAL, not enabled by default

## What "goals" are there?

* `Heartstopper` - Base Hacknet ending, stop PortHack's heart.
* `Veteran` - Gain admin access to a secret post-labs node. (Requires Labyrinths DLC)
* `Completionist` - Achieve both goals listed above.

## What "items" can you unlock in Hacknet?

Every executable is shuffled into the item pool.

## Death Links & Traps

The "Death Link" and only trap in Hacknet is the `ETAS Trap`.

If a user gains this trap, they will immediately be forced into the "Emergency Trace Aversion Sequence." This trap will be queued to fire if "Altitude Loss," the base game sequencer, or CoelTrain recovery are active - so as to not disrupt these events.

## What about the branching missions?

Branching missions will always count the same - as long as the mission is completed, you get the same check.

## What if I don't have Labyrinths?

You can choose to disable Labyrinths content in the settings page. If you have the DLC, it's recommended to play with it!

## Do extensions work?

Extensions will not count towards Archipelago progress. Only the campaign!

## How do I play on the Origin/EA version of Hacknet?

It's impossible, sorry. The DRM used in that version of Hacknet prevents any modifications, and you need that to run the client.
